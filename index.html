Тренинг по HTML и CSS

Толстой Л. Н. Знание - орудие, а не цель.

Сегодня, 28 авгута стартовал тренинг по HTML и CSS .


Цель тренинга – систематизация знаний по указанным технологиям.

Сейчас тренинг проходит в тестовом режиме, поэтому массовый набор на тренинг не объявлялся. В случае положительных отзывов данный тренинг будет проводится на регулярной основе.

Уже сейчас вы можете в комментариях оставлять заявки на участие, а также свои пожелания по программе тренинга (первая версия программы добавлена к данному сообщению) .

Программа тренинга

  1. Обзор тренинга, рекомендуемая литература
    • HTML – общая характеристика, версии HTML
    • Инструменты редактирования HTML, валидация HTML-документа
    • Синтаксис HTML - общие понятия
    • Спецсимволы в HTML-документе
    • Сравнение HTML и XML. XHTML
    • Общие атрибуты элементов - категории, описание, назначение.
    • Атрибуты, описывающие события
    • Базовая структура HTML-документа, "HTML5 Boilerplate"
  2. Структрные элементы страницы в HTML5
    • Форматирование текста (абзацы, разрывы, заголовки, шрифты)
    • Списки
    • Таблицы
    • Изображения
    • Ссылки и якоря
    • Общие атрибуты элементов - категории, описание, назначение.
    • Атрибуты, описывающие события
    • Базовая структура HTML-документа, "HTML5 Boilerplate"
  3. HTML-формы
    • Элемент input, его разновидности
    • Элемент textarea
    • Списки выбора (элемент select)
    • Кнопка button
    • Новые элементы форм в HTML5 - описание, достоинства, проблемы
    • Общие атрибуты элементов - категории, описание, назначение.
    • Группировка элементов в форме
